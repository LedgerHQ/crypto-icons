# 🪙 CryptoIcon

## Introduction

React Native version of `CryptoIcon`, built on Lumen `MediaImage` and `DotSymbol`.
Resolves the icon URL from the Ledger CDN or CoinGecko, overlays an optional network badge,
and shows a letter fallback when no URL resolves.

## Anatomy

<Canvas of={CryptoIconStories.Base} />

- **`ledgerId` (required)**: Ledger internal identifier, e.g. `"bitcoin"` or `"ethereum/erc20/usdc"`
- **`ticker` (required)**: Ticker symbol (e.g. `"BTC"`). First letter is used as the fallback letter when no icon resolves.
- **`size` (optional)**: Icon size in pixels — `12 | 16 | 20 | 24 | 32 | 40 | 48 | 56 | 64` (default `48`)
- **`shape` (optional)**: `"circle"` (default) or `"square"`
- **`network` (optional)**: Ledger ID of the chain to show as a badge overlay
- **`badgePosition` (optional)**: Corner of the badge — `"bottom-end"` by default

## Implementation

### 1. Install

```bash
# crypto-icons itself
npm install @ledgerhq/crypto-icons

# Peer dependencies — Lumen RN components and themes
npm install @ledgerhq/lumen-ui-rnative @ledgerhq/lumen-design-core
```

### 2. Wrap with ThemeProvider

All Lumen RN components require a `ThemeProvider` at the root of your app.
It internally mounts `LumenStyleSheetProvider` which resolves design tokens:

```tsx
import { ThemeProvider } from '@ledgerhq/lumen-ui-rnative';
import { ledgerLiveThemes } from '@ledgerhq/lumen-design-core';

export function App() {
  return (
    <ThemeProvider themes={ledgerLiveThemes} colorScheme="light">
      {/* your app */}
    </ThemeProvider>
  );
}
```

### 3. Use

```tsx
import CryptoIcon, { type CryptoIconProps } from '@ledgerhq/crypto-icons/native';

<CryptoIcon ledgerId="bitcoin" ticker="BTC" size={56} />
```

### With Network Badge

```tsx
<CryptoIcon
  ledgerId="ethereum/erc20/usdc"
  ticker="USDC"
  network="ethereum"
  badgePosition="bottom-end"
  size={56}
/>
```

### Square Shape

```tsx
<CryptoIcon ledgerId="bitcoin" ticker="BTC" size={56} shape="square" />
```

### Typing a wrapper component

```tsx
import CryptoIcon, { type CryptoIconProps } from '@ledgerhq/crypto-icons/native';

const MyIcon = (props: CryptoIconProps) => (
  <CryptoIcon {...props} />
);
```
