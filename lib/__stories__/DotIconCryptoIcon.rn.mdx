# 🔴 DotCryptoIcon

## Introduction

React Native version — wraps `CryptoIcon` with a Lumen `DotIcon` badge to communicate
transaction or account status. The badge overlays a small icon (with a semantic colour) at
any corner of the wrapped component.

## Anatomy

<Canvas of={DotIconNativeStories.Base} />

- **`icon` (required)**: A Lumen symbol component (e.g. `Check`, `Clock`, `Close`) rendered inside the badge
- **`appearance` (required)**: Semantic badge colour — `"success"` · `"muted"` · `"error"`
- **`size` (optional)**: Dot diameter — `16 | 20 | 24` (default `20`)
- **`pin` (optional)**: Corner placement — `"bottom-end"` by default
- **`shape` (optional)**: `"circle"` (default) or `"square"`
- **`children`**: The component the badge is attached to (typically `CryptoIcon`)

## Implementation

### 1. Install

```bash
# crypto-icons + Lumen RN peer dependencies (≥ 0.1.25)
npm install @ledgerhq/crypto-icons @ledgerhq/lumen-ui-rnative @ledgerhq/lumen-design-core
```

See [CryptoIcon → Implementation](?path=/docs/communication-cryptoicon--docs#implementation) for the ThemeProvider setup step.

### 2. Use

```tsx
import CryptoIcon from '@ledgerhq/crypto-icons/native';
import { DotIcon, mediaImageDotIconSizeMap } from '@ledgerhq/lumen-ui-rnative';
import { Check, Clock, Close } from '@ledgerhq/lumen-ui-rnative/symbols';
```

### Success (confirmed)

```tsx
<DotIcon icon={Check} appearance="success" size={mediaImageDotIconSizeMap[56]}>
  <CryptoIcon ledgerId="bitcoin" ticker="BTC" size={56} />
</DotIcon>
```

### Muted (pending)

```tsx
<DotIcon icon={Clock} appearance="muted" size={mediaImageDotIconSizeMap[56]}>
  <CryptoIcon ledgerId="bitcoin" ticker="BTC" size={56} />
</DotIcon>
```

### Error (failed)

```tsx
<DotIcon icon={Close} appearance="error" size={mediaImageDotIconSizeMap[56]} pin="top-end">
  <CryptoIcon ledgerId="bitcoin" ticker="BTC" size={56} />
</DotIcon>
```
